Block disposable email addresses at signup in WooCommerce.

== Description ==

Includes user defined whitelist & blacklists.

Contribute to the plugin on Github: https://github.com/Pross/woo-disposable-emails

The main blocklist is located here: https://github.com/martenson/disposable-email-domains

Settings for the plugin can be found in the main WooCommerce settings panel under Integrations.
